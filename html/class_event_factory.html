Event* EventFactory::createEvent ( EventConfig *  config)

Creates a concrete Event object according to the actual type of the config argument. Two types of events are implemented: CellIDEvent and CellIDTAEvent. CellIDEvent contains the ID of the cell where a mobile device was detected at a time instant, the network type and the timestamp of the event and the CellIDTAEvent additionally contains the timing advance value of the event.

Returns
a pointer to a concrete Event object created according to the actual type of the of the config argument. The two types of event objects supported are:
  • CellIDEvent and in this case the config parameter should point to a CellIDEventconfig object
  • CellIDTAEvent and in this case the config parameter should point to a CellIDTAEventconfig object.
