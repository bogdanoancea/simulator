Detailed Description

This class encapsulated the parameters needed by the factory class to build a CellIDEvent object.

Constructor of the class. Builds a CellIDEventConfig object.

Parameters
tthe timestamp of the event. This is the time measured by the simulation clock.
cellIDthe id of the Antenna object which generated the event.
code- the event code. It can take one of the following values:
  • EventCode::ATTACH_DEVICE
  • EventCode::DETACH_DEVICE
  • EventCode::ALREADY_ATTACHED_DEVICE
  • EventCode::IN_RANGE_NOT_ATTACHED_DEVICE
deviceIDthe id of the mobile device which generated the event.
netTypethe network type (a parameter of the antenna). It can take one of the following values:
  • NetworkType::_3G
  • NetworkType::_4G
