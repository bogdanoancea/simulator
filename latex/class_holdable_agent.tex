\doxysubsection{Detailed Description}
This is the superclass for all agents that represent a device (\mbox{\hyperlink{class_mobile_phone}{Mobile\+Phone}} or \mbox{\hyperlink{class_tablet}{Tablet}}), can by held by a person and can interact with the network. The interaction is triggered every time the location of this agent is set.
