\doxysubsection{Detailed Description}
This is the base class of the \mbox{\hyperlink{class_event_config}{Event\+Config}} objects. It defines the interface that all derived classes should implement. An \mbox{\hyperlink{class_event_config}{Event\+Config}} object is used by the \mbox{\hyperlink{class_event_factory}{Event\+Factory}} to build a specific \mbox{\hyperlink{class_event}{Event}} object and it contains the parameters of the specific event.
