\doxysubsection{Detailed Description}
This class encapsulates the information that need to be saved when a network event is generated by the interaction between a mobile device and the network. The \char`\"{}\+Cell I\+D\char`\"{} event contains the timestamp of the event, the cell id (antenna id), the event code (\begin{DoxySeeAlso}{See also}
\mbox{\hyperlink{_event_type_8h_a2628ea8d12e8b2563c32f05dc7fff6fa}{Event\+Type}} enumeration for the possible values), the device id and the network type (one of the \mbox{\hyperlink{_network_type_8h_a3a159600500d5d7248be5bd1ca1f8d83ac88ee8f01394dbabf5c4487986d04971}{Network\+Type\+::\+\_\+3G}} or \mbox{\hyperlink{_network_type_8h_a3a159600500d5d7248be5bd1ca1f8d83ae5ee9782bb207b843ba97eaa2d0fe26b}{Network\+Type\+::\+\_\+4G}} values). 
\end{DoxySeeAlso}
